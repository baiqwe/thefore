# 广告禁用说明

## 问题说明

Adsterra 广告服务目前返回 403 错误，导致控制台显示网络错误。这是浏览器网络层面的错误，无法通过代码完全隐藏。

## 解决方案

### 方案 1：暂时禁用广告（推荐）

如果需要暂时禁用广告以避免控制台错误，可以设置环境变量：

在 `.env.local` 文件中添加：
```
NEXT_PUBLIC_ADS_ENABLED=false
```

或者在 Cloudflare Pages 的环境变量中设置：
- Key: `NEXT_PUBLIC_ADS_ENABLED`
- Value: `false`

### 方案 2：等待服务恢复正常

403 错误通常是因为：
- 广告服务暂时不可用
- 广告配置有问题
- IP 地址被限制

等待服务恢复正常后，错误会自动消失。

### 方案 3：联系 Adsterra 支持

如果 403 错误持续存在，可能需要：
1. 检查 Adsterra 账户状态
2. 验证广告 key 是否正确
3. 联系 Adsterra 技术支持

## 代码说明

代码已经添加了错误处理：
- 广告加载失败时自动隐藏广告容器
- 不会影响页面其他功能
- 不会输出额外的警告信息

但是浏览器网络层面的 403 错误仍然会在控制台显示，这是正常现象。
