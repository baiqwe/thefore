## 🔑 核心功能

1. **图片管理**
   - 统一路径管理
   - 多优先级图片源
   - 自动错误处理

2. **自动提取**
   - YouTube 视频搜索
   - 视频下载和处理
   - 关键帧提取

3. **智能关联**
   - 图片与章节自动关联
   - 数据自动更新

4. **UI 组件**
   - 封面图组件
   - 章节图片组件

## 🛠️ 技术栈

- **前端**: Next.js 14 + TypeScript + React
- **工具**: yt-dlp (视频下载), ffmpeg (视频处理)
- **语言**: Node.js (脚本), TypeScript (前端)

## 📋 快速开始

基于这些文档开发时：

1. 安装依赖工具（yt-dlp, ffmpeg）
2. 创建目录结构
3. 实现核心模块（按技术文档）
4. 开发自动化脚本
5. 集成 UI 组件
6. 测试和优化
